All you ubuntu fans the new version of Ubuntu is coming soon.

Stable version of Ubuntu 10 which is Ubuntu 10.04 LTS will be released on the 29th of April 2010

If you are eager to grab the latest version you can always use 'sudo update-manager -d' from the ubuntu shell to upgrade to the beta version 1 even now.

When you execute this command there will be an option appearing at the top of the update manager with the upgrade link.

But its always better to wait for the stable version as there is only about three weeks remaining from today(3rd April 2010) onwards.

Visit the following to get more information on the current beta releases.
http://www.ubuntu.com/testing/lucid/beta1

Or go to the official release page at: http://releases.ubuntu.com/releases/10.04/
